# Wackee 怪话记录器 🎙️

一个 [AstrBot](https://github.com/AstrBotDevs/AstrBot) 插件，用于记录群友的怪话、统计排行并随机发送。

## ✨ 功能

### 📝 记录怪话
引用/回复一条群友的消息，然后发送 `记录`，即可将该消息保存为怪话记录。

- 按 **群号 → QQ号** 分层存储
- 自动记录消息内容、时间、记录人
- 重复记录会累计出现次数，不新增唯一条目

### 📢 来句怪话
- `来句怪话 @某人` — 随机发送该用户被记录过的一条怪话
- `来句怪话` — 从本群所有记录中随机抽取一条怪话
- 每次发送自动计数

### 🏆 怪话排行
- `怪话排行` — 显示当前群里怪话累计出现次数最多的人
- 若有并列榜首，会一起显示
- 同时展示榜首用户出现最多次的怪话；如果该条记录带图，会连图片一起发送

### 帮助
- `怪话帮助` — 显示所有指令的功能、用法和注意事项

### ⚙️ 配置项

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `cross_group_search` | bool | `false` | 启用后，@某人时若本群无记录则搜索其他群 |

## 📦 安装

在 AstrBot WebUI 的插件管理中，通过仓库地址安装：

```
https://github.com/storyAura/astrbot_plugin_Wackee
```

## 💡 使用示例

```
用户A: 今天天气真好啊，好想摸鱼
用户B: [引用用户A的消息] 记录
机器人: ✅ 已记录 用户A 的怪话！
       📈 该条怪话当前已出现 1 次

用户B: [再次引用用户A的同一条消息] 记录
机器人: ✅ 已记录 用户A 的怪话！
       📈 该条怪话当前已出现 2 次

用户C: 来句怪话 @用户A
机器人: 📢 用户A 曾说过：
       「今天天气真好啊，好想摸鱼」
       —— 2026-04-08 20:00:00（已被发送 1 次）

用户D: 怪话排行
机器人: 当前群聊怪话第一
       1. 用户A
       累计出现：2 次
       唯一记录：1 条
       代表怪话：今天天气真好啊，好想摸鱼（出现 2 次）

用户E: 怪话帮助
机器人: Wackee 指令帮助
       1. 记录
       功能：保存一条被引用的群消息，支持文本和图片
       用法：先引用或回复一条消息，再发送“记录”
       ...
```

## ⚠️ 注意事项

- **仅限群聊**使用
- 被引用的消息需要是机器人在线期间发送的（插件会缓存最近的群聊消息）
- 数据存储在 `data/plugin_data/astrbot_plugin_wackee/` 目录下，插件更新不会丢失数据
